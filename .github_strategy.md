# GitHub Strategy — SecureRAG Guard

## Структура репозиториев

| Репо | Видимость | Содержит | Для кого |
|------|-----------|----------|----------|
| `mastoodont/Web4QA` (этот) | **Public** | Код + заглушки движка | Партнёры, разработчики, интеграторы |

## Шаг 1 — Первая загрузка на GitHub (замена старого проекта)

Выполни на своём компьютере в папке `C:\Users\Dell\rag\deploy_v3`:

```powershell
# Перейди в папку с проектом
cd C:\Users\Dell\rag\deploy_v3

# Инициализируй git если ещё нет
git init

# Подключи удалённый репозиторий
git remote remove origin 2>$null
git remote add origin https://github.com/mastoodont/Web4QA.git

# ВАЖНО: замени 4 закрытых файла на заглушки из этого архива
# (файлы уже подготовлены — просто скопируй их поверх)
# app/core/security_engine.py   → stub
# app/core/scoring.py           → stub
# app/ml/classifier.py          → stub
# app/ml/training_data.py       → stub

# Добавь все файлы
git add .

# Первый коммит
git commit -m "feat: SecureRAG Guard — real-time security API for LLM/RAG pipelines"

# Принудительно замени старый контент репозитория
git push --force origin main
```

> ⚠️ `--force` нужен только один раз, чтобы заменить старый проект.
> После этого используй обычный `git push origin main`.

## Шаг 3 — Локальная работа с полным кодом

Твой реальный полный проект остаётся ТОЛЬКО локально:
`C:\Users\Dell\rag\deploy_v3`

Никогда не пуши `security_engine.py`, `scoring.py`, `classifier.py`, `training_data.py`
с реальным кодом в публичный репо.

Чтобы защититься от случайного пуша — добавь в `.gitignore` (опционально):
```
# Эти файлы никогда не идут в публичный репо с реальным кодом
# app/core/security_engine.py
# app/core/scoring.py
# app/ml/classifier.py
# app/ml/training_data.py
```

## Шаг 4 — Деплой на Railway (из полного локального кода)

Railway деплоится напрямую из локальной папки или приватного репо — не из этого публичного:

```powershell
# Установи Railway CLI
npm install -g @railway/cli

# Логин
railway login

# Деплой из локальной папки (с полным кодом)
cd C:\Users\Dell\rag\deploy_v3
railway up
```

Или через приватный репо на GitHub:
1. Создай приватный репо `mastoodont/Web4QA-private`
2. Запушь туда полный код: `git push private-origin main`
3. В Railway подключи именно приватный репо
